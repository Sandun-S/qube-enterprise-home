Quick Start Guide

1. Set Cloud API URL → Save & Test.
2. Register / Login — create your org (or run docker compose -f docker-compose.dev.yml up -d).
3. Claim Qube — enter Q-1001, copy the token shown.
4. Set QUBE_TOKEN in docker-compose.dev.yml for conf-agent and enterprise-influx-to-sql, then docker compose up -d.
5. Templates → migration 002 seeds Schneider PM5100, Generic MQTT, APC UPS.
6. Add Gateway → select qube + protocol + host.
7. Add Sensor → select gateway + template + address params → CSV auto-generated.
8. Conf-agent syncs within 30s, docker compose updates on the Qube.
9. When data flows through core-switch → InfluxDB → enterprise-influx-to-sql → TP-API → Postgres.
10. Telemetry → paste sensor UUID → query readings.
